# JavaScript Data Types

## What I Learned Today

JavaScript has different types of data.

### 1. String
Text values.
Example:
"Hello"

### 2. Number
Numeric values.
Example:
10, 3.14

### 3. Boolean
True or False values.

### 4. Undefined
A variable declared but not assigned a value.

### 5. Null
Represents an empty value intentionally.

## My Understanding

Data types define what kind of value a variable can store.
