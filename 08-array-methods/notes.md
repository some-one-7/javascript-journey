# JavaScript Array Methods

## What I Learned Today

Arrays have built-in methods to modify and work with data.

### shift()
Removes the first element of the array.

### unshift()
Adds a new element at the beginning of the array.

### includes()
Checks if an element exists in the array.

### indexOf()
Returns the index position of an element.

### slice()
Returns a portion of the array without modifying the original array.

### concat()
Combines two arrays into one.

### join()
Converts an array into a string.

### reverse()
Reverses the order of elements in an array.

### sort()
Sorts elements of an array.

## My Understanding

Array methods make it easier to manipulate and manage collections of data.
