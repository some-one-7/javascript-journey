# JavaScript DOM Introduction

## What I Learned Today

DOM stands for Document Object Model.

It allows JavaScript to interact with HTML elements.

### document.getElementById()

This method selects an HTML element using its id.

Example:
document.getElementById("title")

### innerText

Used to change the text inside an HTML element.

Example:
document.getElementById("title").innerText = "Hello JavaScript"

## My Understanding

The DOM connects JavaScript with HTML and allows dynamic changes on webpages.
