# Day 27 - Advanced To-Do App

## What I Built

A full-featured To-Do app with:
- Add task
- Delete task
- Mark complete
- Filter (All / Completed / Pending)
- LocalStorage support

---

## Concepts Used

### 1. CRUD Operations
- Create → Add task
- Read → Show tasks
- Update → Toggle complete
- Delete → Remove task

---

### 2. localStorage
Saved tasks in browser

---

### 3. Event Delegation
Handled clicks using parent element

---

### 4. Filtering
Displayed tasks based on status

---

## Key Learning

"Real apps = CRUD + state + UI + storage"
