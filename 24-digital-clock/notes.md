# Day 24 - Digital Clock

## What I Built

A real-time digital clock:
- Shows current time
- Updates every second

---

## Concepts Used

### 1. Date Object
Used to get current time

Example:
new Date()

---

### 2. setInterval()
Runs function every 1 second

---

### 3. Formatting Time
Added leading zero for single digits

---

### 4. DOM Update
Updated time using innerText

---

## Key Learning

"JavaScript can handle real-time updates using timers"
