# Day 31 – Search & Sort System

## Concepts Learned
- Searching with string matching (includes)
- Sorting arrays using sort()
- Combining multiple filters
- Copying arrays using spread operator (...)

## Key Learnings
- Always copy array before sorting (avoid mutating state)
- Search should be case-insensitive
- Sorting logic depends on user selection

## Features
- Live search (real-time filtering)
- Sort by date (new/old)
- Sort by amount
- Dynamic UI update
