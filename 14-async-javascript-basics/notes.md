# Day 14 - Async JavaScript Basics

## What I Learned

### 1. setTimeout()
- Runs code after a delay
- Example:
  setTimeout(() => {
    console.log("Hello");
  }, 2000);

---

### 2. setInterval()
- Runs code repeatedly after a fixed time

---

### 3. clearInterval()
- Stops setInterval

---

### 4. clearTimeout()
- Stops setTimeout

---

### 5. Sync vs Async

#### Sync:
- Code runs line by line

#### Async:
- Code runs after delay without blocking

---

## Features Built

- Timer (1 to 10)
- Alert after 3 seconds
- Interval start/stop

---

## Key Concept

"JavaScript does not wait, it schedules tasks"
