# JavaScript Conditionals

## What I Learned Today

Conditionals allow JavaScript to make decisions.

### if Statement
Used to run code only if a condition is true.

Example:
if (age >= 18) {
  console.log("Adult");
}

### if-else Statement
Runs one block if the condition is true, otherwise another block.

### else-if
Used when multiple conditions need to be checked.

## My Understanding

Conditionals help control the flow of a program by executing different code based on conditions.
