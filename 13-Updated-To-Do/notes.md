# 📝 To-Do App (Learning Project)

This is a simple To-Do List app built using **HTML, CSS, and JavaScript**.
I made this project to practice basic frontend concepts.

---

## 🚀 Features

* Add new tasks
* Mark tasks as completed
* Edit tasks
* Delete tasks
* Data is saved using localStorage (even after refresh)

---

## 🧠 What I Learned

* How to manipulate the DOM
* Handling events (click, keypress)
* Storing data in arrays
* Using localStorage
* Updating UI dynamically

---

## 🔄 How It Works

1. User enters a task
2. Task is stored in an array
3. Array is saved in localStorage
4. UI is updated
5. Any change → save again + re-render

---

## 📂 Files

* `index.html` → structure
* `style.css` → styling
* `todo.js` → functionality

---

## ▶️ How to Run

Just open `index.html` in your browser.
